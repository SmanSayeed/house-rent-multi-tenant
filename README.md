House rental management application

How to run the project?
project setup
Clone the repository
composer update
php artisan migrate:fresh --seed
php artisan serve
project will run in : http://localhost:8000
Set email credentials from mailtrap

Home page: http://localhost:8000
Admin login:
http://localhost:8000/admin/login

password: password
